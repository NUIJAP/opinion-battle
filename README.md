# 立場 BATTLE

社会的テーマでAIと本気の議論対決をするスマホゲーム。

## 概要

- **ターゲット**: 20-40代のスマホユーザー（特に日本人）
- **利用シーン**: 通勤・待ち時間・就寝前の「暇つぶし」
- **プレイ時間**: 1プレイ 60-180秒
- **核となる体験**: AIとの本気の議論対決が見られる

## 実装予定

## 技術スタック

- **フロント**: Next.js 14+ (React, TypeScript)
- **スタイリング**: Tailwind CSS
- **AI**: Claude API (Anthropic)
- **DB**: Supabase (PostgreSQL)
- **外部API**: NewsAPI / GoogleNews API
- **通知**: Firebase Cloud Messaging
- **ホスティング**: Vercel
- **デスクトップ**: Cowork (複雑ロジック)
